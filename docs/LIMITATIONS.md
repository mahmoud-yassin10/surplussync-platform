# Limitations

- Current training data is synthetic.
- The official demo scenario is deterministic and contract-locked.
- Shortage probability is heuristic for general predictions.
- The preparation policy is safety-oriented, not a full newsvendor optimizer.
- Real deployment performance has not been demonstrated.
- No private student-level data is used in the prototype.
- There is no production identity provider.
- There is no durable enterprise database.
- Carbon estimates are not audited.
- The app should not be described as production-ready.
- The project does not claim real Chicago Public Schools results.
