# Third-Party Notices

This file lists major dependencies and methodologies used by the SurplusSync Plus snapshot. It is not a full license inventory.

- React and React DOM for UI rendering.
- TanStack Start, TanStack Router, and TanStack Query for the frontend app framework and routing.
- Vite for local development and production builds.
- Tailwind CSS for styling.
- Lucide React for icons.
- MapLibre GL JS for the recovery map.
- CARTO basemap tiles for tokenless map display.
- FastAPI and Uvicorn for the ML HTTP service.
- CatBoost for tabular regression models.
- pandas, NumPy, and scikit-learn for data processing and evaluation.
- Pydantic and Zod for schema validation.
- Vitest, pytest, ruff, mypy, TypeScript, and ESLint for validation.
- Google Gemini integration is optional for language generation; deterministic fallback supports local demos without a key.
- ReFED methodology is referenced for food-waste recovery and climate-impact assumptions.

Do not treat this file as legal advice. A complete dependency license review is still required before production distribution.
